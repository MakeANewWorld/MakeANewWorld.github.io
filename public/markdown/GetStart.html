安裝 JetBrains Runtime（DCEVM 用）、Fabric（開發環境）、JDK 21（含 OpenGL 庫）與 IntelliJ IDEA

本指南適用於完全沒有電腦基礎的使用者，並且專門針對 Windows 平台。

1. 安裝 JDK 21（Microsoft 提供）

Microsoft 的 JDK 內建 OpenGL 庫，適合我們的需求。

  1. 下載 JDK 21：

    • 進入 Microsoft OpenJDK 官網。
    • 找到 Windows x64 版本的 JDK 21，下載 .msi 安裝檔。

  2. 安裝 JDK：

    • 執行剛剛下載的 .msi 檔案。
    • 一路點擊 Next，直到安裝完成。
    • 記住安裝路徑（預設為 C:\Program Files\Microsoft\jdk-21）。

  3. 設定環境變數：

    • 按 Win + S 搜尋 環境變數，選擇 編輯系統環境變數。
    • 在 系統變數 中找到 Path，點擊 編輯。
    • 點擊 新增，輸入 C:\Program Files\Microsoft\jdk-21\bin。
    • 點擊 確定 保存設定。

  4. 確認 JDK 是否成功安裝：

    • 按 Win + R，輸入 cmd，按 Enter。
    • 輸入以下指令檢查 JDK 版本：
      java -version
    • 你應該會看到類似 openjdk version "21" 的輸出。

2. 安裝 JetBrains Runtime（DCEVM）

DCEVM 允許在 Java 進程運行時重新載入修改過的類，而 JetBrains Runtime 提供了最相容的環境。

  1. 下載 JetBrains Runtime：

    • 進入 JetBrains Runtime 下載頁面。
    • 找到最新的 JBR 17（DCEVM） 版本，下載 Windows x64 的 .zip 檔案。

  2. 解壓縮並設定：

    • 把 .zip 解壓縮到 C:\JetBrains\jbr_dcevm（可自行選擇路徑）。
    • 在 環境變數 中新增 JAVA_HOME，值設為 C:\JetBrains\jbr_dcevm。
    • 編輯 Path 變數，新增 C:\JetBrains\jbr_dcevm\bin。

  3. 確認 JBR 是否成功安裝：

    • 打開 cmd，輸入：
      java -version
    • 你應該會看到 JetBrains Runtime 相關訊息。

3. 安裝 Fabric（開發環境）

  1. 安裝 Fabric Loom：

    • 下載 Gradle 並安裝。
    • 下載 Fabric Loom 並設定專案。

  2. 建立 Fabric 開發專案：

    • 在 cmd 中執行：
      git clone https://github.com/FabricMC/fabric-example-mod.git
cd fabric-example-mod
./gradlew build

  3. 確認環境是否可用：

    • 檢查 build/libs 資料夾是否有 .jar 文件。

4. 安裝 IntelliJ IDEA

  1. 下載並安裝 IDEA：

    • 進入 IntelliJ IDEA 官網。
    • 下載 Community（免費版）或 Ultimate（付費版）。
    • 執行 .exe 安裝程式，選擇 JetBrains Runtime 作為預設 JDK。

  2. 設定 Fabric 專案：

    • 在 IDEA 中打開 fabric-example-mod。
    • 設定 Gradle JVM 為 JetBrains Runtime。
    • 執行 ./gradlew genIntelliJRuns 來生成執行設定。

5. 總結

  • JDK 21（Microsoft）：提供 OpenGL 支援。
  • JetBrains Runtime（DCEVM）：允許熱加載。
  • Fabric 開發環境：用於 Minecraft Mod 開發。
  • IntelliJ IDEA：作為開發工具。

到這裡，你的環境就準備好了！🚀 `
